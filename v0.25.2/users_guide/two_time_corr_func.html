Two-time Correlation Functions ‚Äã

Introduction ‚Äã

With the QuantumToolbox.jl time-evolution function mesolve, a state vector (Ket) or density matrix (Operator) can be evolved from an initial state at t0 to an arbitrary time t, namely

œÅ^(t)=G(t,t0){œÅ^(t0)},

where G(t,t0){‚ãÖ} is the propagator defined by the equation of motion. The resulting density matrix can then be used to evaluate the expectation values of arbitrary combinations of same-time operators.

To calculate two-time correlation functions on the form ‚ü®A^(t+œÑ)B^(t)‚ü©, we can use the quantum regression theorem [see, e.g., Gardiner and Zoller (2004)] to write

‚ü®A^(t+œÑ)B^(t)‚ü©=Tr[A^G(t+œÑ,t){B^œÅ^(t)}]=Tr[A^G(t+œÑ,t){B^G(t,0){œÅ^(0)}}],

We therefore first calculate œÅ^(t)=G(t,0){œÅ^(0)} using mesolve with œÅ^(0) as initial state, and then again use mesolve to calculate G(t+œÑ,t){B^œÅ^(t)} using B^œÅ^(t) as initial state.

Note that if the initial state is the steady state, then œÅ^(t)=G(t,0){œÅ^ss}=œÅ^ss and

‚ü®A^(t+œÑ)B^(t)‚ü©=Tr[A^G(t+œÑ,t){B^œÅ^ss}]=Tr[A^G(œÑ,0){B^œÅ^ss}]=‚ü®A^(œÑ)B^(0)‚ü©,

which is independent of t, so that we only have one time coordinate œÑ.

QuantumToolbox.jl provides a family of functions that assists in the process of calculating two-time correlation functions. The available functions and their usage is shown in the table below.

Function callCorrelation function
correlation_2op_2t‚ü®A^(t+œÑ)B^(t)‚ü© or ‚ü®A^(t)B^(t+œÑ)‚ü©
correlation_2op_1t‚ü®A^(œÑ)B^(0)‚ü© or ‚ü®A^(0)B^(œÑ)‚ü©
correlation_3op_1t‚ü®A^(0)B^(œÑ)C^(0)‚ü©
correlation_3op_2t‚ü®A^(t)B^(t+œÑ)C^(t)‚ü©

The most common used case is to calculate the two time correlation function ‚ü®A^(œÑ)B^(0)‚ü©, which can be done by correlation_2op_1t.

Steadystate correlation function ‚Äã

The following code demonstrates how to calculate the ‚ü®x^(t)x^(0)‚ü© correlation for a leaky cavity with three different relaxation rates Œ≥.

julia
tlist = LinRange(0, 10, 200)
a = destroy(10)
x = a' + a
H = a' * a

# if the initial state is specified as `nothing`, the steady state will be calculated and used as the initial state.
corr1 = correlation_2op_1t(H, nothing, tlist, [sqrt(0.5) * a], x, x)
corr2 = correlation_2op_1t(H, nothing, tlist, [sqrt(1.0) * a], x, x)
corr3 = correlation_2op_1t(H, nothing, tlist, [sqrt(2.0) * a], x, x)

# plot by CairoMakie.jl
fig = Figure(size = (500, 350))
ax = Axis(fig[1, 1], xlabel = L"Time $t$", ylabel = L"\langle \hat{x}(t) \hat{x}(0) \rangle")
lines!(ax, tlist, real(corr1), label = L"\gamma = 0.5", linestyle = :solid)
lines!(ax, tlist, real(corr2), label = L"\gamma = 1.0", linestyle = :dash)
lines!(ax, tlist, real(corr3), label = L"\gamma = 2.0", linestyle = :dashdot)

axislegend(ax, position = :rt)

fig

Emission spectrum ‚Äã

Given a correlation function ‚ü®A^(œÑ)B^(0)‚ü©, we can define the corresponding power spectrum as

S(œâ)=‚à´‚àí‚àû‚àû‚ü®A^(œÑ)B^(0)‚ü©e‚àíiœâœÑdœÑ

In QuantumToolbox.jl, we can calculate S(œâ) using either spectrum, which provides several solvers to perform the Fourier transform semi-analytically, or we can use the function spectrum_correlation_fft to numerically calculate the fast Fourier transform (FFT) of a given correlation data.

The following example demonstrates how these methods can be used to obtain the emission (A^=a^‚Ä† and B^=a^) power spectrum.

julia
N = 4             # number of cavity fock states
œâc = 1.0 * 2 * œÄ  # cavity frequency
œâa = 1.0 * 2 * œÄ  # atom frequency
g  = 0.1 * 2 * œÄ  # coupling strength
Œ∫  = 0.75         # cavity dissipation rate
Œ≥  = 0.25         # atom dissipation rate

# Jaynes-Cummings Hamiltonian
a  = tensor(destroy(N), qeye(2))
sm = tensor(qeye(N), destroy(2))
H = œâc * a' * a + œâa * sm' * sm + g * (a' * sm + a * sm')

# collapse operators
n_th = 0.25
c_ops = [
    sqrt(Œ∫ * (1 + n_th)) * a,
    sqrt(Œ∫ *      n_th)  * a',
    sqrt(Œ≥)              * sm,
];

# calculate the correlation function using mesolve, and then FFT to obtain the spectrum.
# Here we need to make sure to evaluate the correlation function for a sufficient long time and
# sufficiently high sampling rate so that FFT captures all the features in the resulting spectrum.
tlist = LinRange(0, 100, 5000)
corr = correlation_2op_1t(H, nothing, tlist, c_ops, a', a; progress_bar = Val(false))
œâlist1, spec1 = spectrum_correlation_fft(tlist, corr)

# calculate the power spectrum using spectrum
# using Exponential Series (default) method
œâlist2 = LinRange(0.25, 1.75, 200) * 2 * œÄ
spec2 = spectrum(H, œâlist2, c_ops, a', a; solver = ExponentialSeries())

# calculate the power spectrum using spectrum
# using Pseudo-Inverse method
spec3 = spectrum(H, œâlist2, c_ops, a', a; solver = PseudoInverse())

# plot by CairoMakie.jl
fig = Figure(size = (500, 350))
ax = Axis(fig[1, 1], title = "Vacuum Rabi splitting", xlabel = "Frequency", ylabel = "Emission power spectrum")
lines!(ax, œâlist1 / (2 * œÄ), spec1, label = "mesolve + FFT", linestyle = :solid)
lines!(ax, œâlist2 / (2 * œÄ), spec2, label = "Exponential Series", linestyle = :dash)
lines!(ax, œâlist2 / (2 * œÄ), spec3, label = "Pseudo-Inverse", linestyle = :dashdot)

xlims!(ax, œâlist2[1] / (2 * œÄ), œâlist2[end] / (2 * œÄ))
axislegend(ax, position = :rt)

fig

Non-steadystate correlation function ‚Äã

More generally, we can also calculate correlation functions of the kind ‚ü®A^(t1+t2)B^(t1)‚ü©, i.e., the correlation function of a system that is not in its steady state. In QuantumToolbox.jl, we can evaluate such correlation functions using the function correlation_2op_2t. The default behavior of this function is to return a matrix with the correlations as a function of the two time coordinates (t1 and t2).

julia
t1_list = LinRange(0, 10.0, 200)
t2_list = LinRange(0, 10.0, 200)

N = 10
a = destroy(N)
x = a' + a
H = a' * a

c_ops = [sqrt(0.25) * a]

Œ± = 2.5
œÅ0 = coherent_dm(N, Œ±)

corr = correlation_2op_2t(H, œÅ0, t1_list, t2_list, c_ops, x, x; progress_bar = Val(false))

# plot by CairoMakie.jl
fig = Figure(size = (500, 400))

ax = Axis(fig[1, 1], title = L"\langle \hat{x}(t_1 + t_2) \hat{x}(t_1) \rangle", xlabel = L"Time $t_1$", ylabel = L"Time $t_2$")

heatmap!(ax, t1_list, t2_list, real(corr))

fig

Example: first-order optical coherence function ‚Äã

This example demonstrates how to calculate a correlation function on the form ‚ü®A^(œÑ)B^(0)‚ü© for a non-steady initial state. Consider an oscillator that is interacting with a thermal environment. If the oscillator initially is in a coherent state, it will gradually decay to a thermal (incoherent) state. The amount of coherence can be quantified using the first-order optical coherence function

g(1)(œÑ)=‚ü®a^‚Ä†(œÑ)a^(0)‚ü©‚ü®a^‚Ä†(œÑ)a^(œÑ)‚ü©‚ü®a^‚Ä†(0)a^(0)‚ü©.

For a coherent state |g(1)(œÑ)|=1, and for a completely incoherent (thermal) state g(1)(œÑ)=0. The following code calculates and plots g(1)(œÑ) as a function of œÑ:

julia
œÑlist = LinRange(0, 10, 200)

# Hamiltonian
N = 15
a = destroy(N)
H = 2 * œÄ * a' * a

# collapse operator
G1 = 0.75
n_th = 2.00  # bath temperature in terms of excitation number
c_ops = [
    sqrt(G1 * (1 + n_th)) * a,
    sqrt(G1 *      n_th)  * a'
]

# start with a coherent state of Œ± = 2.0
œÅ0 = coherent_dm(N, 2.0)

# first calculate the occupation number as a function of time
n = mesolve(H, œÅ0, œÑlist, c_ops, e_ops = [a' * a], progress_bar = Val(false)).expect[1,:]
n0 = n[1] # occupation number at œÑ = 0

# calculate the correlation function G1 and normalize with n to obtain g1
g1 = correlation_2op_1t(H, œÅ0, œÑlist, c_ops, a', a, progress_bar = Val(false))
g1 = g1 ./ sqrt.(n .* n0)

# plot by CairoMakie.jl
fig = Figure(size = (500, 350))
ax = Axis(fig[1, 1], title = "Decay of a coherent state to an incoherent (thermal) state", xlabel = L"Time $\tau$")
lines!(ax, œÑlist, real(g1), label = L"g^{(1)}(\tau)", linestyle = :solid)
lines!(ax, œÑlist, real(n), label = L"n(\tau)", linestyle = :dash)

axislegend(ax, position = :rt)

fig

Example: second-order optical coherence function ‚Äã

The second-order optical coherence function, with time-delay œÑ, is defined as

g(2)(œÑ)=‚ü®a^‚Ä†(0)a^‚Ä†(œÑ)a^(œÑ)a^(0)‚ü©‚ü®a^‚Ä†(0)a^(0)‚ü©2.

For a coherent state g(2)(œÑ)=1, for a thermal state g(2)(œÑ=0)=2 and it decreases as a function of time (bunched photons, they tend to appear together), and for a Fock state with n-photons g(2)(œÑ=0)=n(n‚àí1)/n2<1 and it increases with time (anti-bunched photons, more likely to arrive separated in time).

To calculate this type of correlation function with QuantumToolbox.jl, we can use correlation_3op_1t, which computes a correlation function on the form ‚ü®A^(0)B^(œÑ)C^(0)‚ü© (three operators and one delay-time vector). We first have to combine the central two operators into one single one as they are evaluated at the same time, e.g. here we do B^(œÑ)=a^‚Ä†(œÑ)a^(œÑ)=(a^‚Ä†a^)(œÑ).

The following code calculates and plots g(2)(œÑ) as a function of œÑ for a coherent, thermal and Fock state:

julia
œÑlist = LinRange(0, 25, 200)

# Hamiltonian
N = 25
a = destroy(N)
H = 2 * œÄ * a' * a

Œ∫ = 0.25
n_th = 2.0  # bath temperature in terms of excitation number
c_ops = [
    sqrt(Œ∫ * (1 + n_th)) * a,
    sqrt(Œ∫ *      n_th)  * a'
]

cases = [
    Dict("state" => coherent_dm(N, sqrt(2)), "label" => "coherent state", "lstyle" => :solid),
    Dict("state" => thermal_dm(N, 2), "label" => "thermal state", "lstyle" => :dash),
    Dict("state" => fock_dm(N, 2), "label" => "Fock state", "lstyle" => :dashdot),
]

# plot by CairoMakie.jl
fig = Figure(size = (500, 350))
ax = Axis(fig[1, 1], xlabel = L"Time $\tau$", ylabel = L"g^{(2)}(\tau)")

for case in cases
    œÅ0 = case["state"]

    # calculate the occupation number at œÑ = 0
    n0 = expect(a' * a, œÅ0)

    # calculate the correlation function g2
    g2 = correlation_3op_1t(H, œÅ0, œÑlist, c_ops, a', a' * a, a, progress_bar = Val(false))
    g2 = g2 ./ n0^2

    lines!(ax, œÑlist, real(g2), label = case["label"], linestyle = case["lstyle"])
end

axislegend(ax, position = :rt)

fig
